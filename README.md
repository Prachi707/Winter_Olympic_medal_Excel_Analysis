# Winter Olympic Medal Data Excel Analysis

This project analyzes Winter Olympic medal data using Microsoft Excel. I worked with a public dataset, cleaned the data, analyzed it, and created charts, pivot tables, slicers, and a dashboard to make the insights easier to understand.

## Data Source

The dataset was sourced from Kaggle and used for learning and portfolio-building purposes.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Slicers
- Charts
- Data Cleaning

## Project Details

- The dataset contains 9 columns: year, sport, event, country, gender, medal rank, medal type, athlete/team name, and athlete age.
- The dataset has 2,866 rows.
- I removed unnecessary columns, handled null values, and fixed data inconsistencies using Excel formulas, filters, and other data-cleaning tools.
- The analysis focuses on the top 10 countries by medals, gender distribution, sport popularity, and Olympic medal trends over time.
- Slicers were added so viewers can filter the data interactively.

## Key Insights

- Norway had the highest total medal count in the dataset.
- The United States and Germany were also among the top-performing countries.
- Speedskating, Cross-Country Skiing, and Alpine Skiing had the highest number of medal records.
- Men had more medal records than women in this dataset.
- The dataset covers Winter Olympic records from 1924 to 2014.
